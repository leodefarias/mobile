# FIAP Mobile - Portal do Aluno

## Descrição do Projeto

Aplicativo mobile desenvolvido em React Native com Expo, voltado para os alunos da FIAP. O objetivo é oferecer uma plataforma simples e prática para que os alunos acessem informações acadêmicas do dia a dia, como horários de aula, notas, eventos e materiais de apoio — tudo em um único lugar.

Atualmente o projeto conta com a **tela de login** implementada (não-funcional), servindo como ponto de partida para o desenvolvimento das demais funcionalidades.

## Ideia Principal

Criar um app fácil e intuitivo para o aluno da FIAP acompanhar sua vida acadêmica sem complicação. O app centraliza as informações mais importantes — aulas, notas, provas, eventos e materiais — em uma interface limpa com tema escuro e a identidade visual da FIAP.

## Requisitos Funcionais

### Aba Início
- Calendário de aulas de cada dia da semana
- Notificações sobre eventos e tarefas pendentes

### Aba Notas
- Boletim de notas por disciplina
- Registro de faltas

### Aba Agenda
- Calendário anual para eventos acadêmicos
- Calendário de provas e entrega de atividades

### Aba Materiais
- Acesso aos materiais de apoio de cada disciplina

### Aba Perfil
- Visualização da carteirinha digital do aluno

## Estrutura de Abas

| Aba | Funcionalidades |
|---|---|
| **Início** | Aulas do dia, notificações de eventos e tarefas |
| **Notas** | Boletim de notas e registro de faltas |
| **Agenda** | Calendário de eventos, provas e entregas |
| **Materiais** | Materiais de apoio por disciplina |
| **Perfil** | Carteirinha digital do aluno |

## Tecnologias

- **React Native** com **Expo** (SDK 55)
- **TypeScript**
- **@expo/vector-icons** (ícones Feather e Fontisto)
- Tema escuro com cor primária `#ED145B` (rosa FIAP)
